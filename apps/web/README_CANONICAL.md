5. Install dependencies:
   - `pnpm install` or `npm install`
6. Start dev server:
   - `pnpm run dev` or `npm run dev`
   - or use `npm start` which maps to the dev script.
7. To build for production:
   - `pnpm run build`
   - `pnpm run preview` to locally preview the build.
